🎯 Problem Statement

Digital platforms host tens of thousands of titles, yet users often spend more time browsing than watching — a phenomenon called Choice Paralysis.

Simple genre/tag metadata is too broad. A user who loved Interstellar doesn't want just any Sci-Fi film — they want films with similar thematic depth, tone, and narrative style. This project reads actual plot text of every movie to mathematically measure semantic similarity.

🔬 TF-IDF + Cosine Similarity Algorithm
📝 Corpus: Each movie's overview + genres + tagline forms one document. Preprocessed: lowercase → punctuation removed → stopwords → lemmatized
🧮 TF-IDF: TfidfVectorizer(max_features=50000, ngram_range=(1,2)) → sparse matrix (45,447 × 50,000)
📐 Cosine Similarity: Dot product of normalized TF-IDF vectors. Range [0,1]. 0=no overlap, 1=identical
🏆 Ranking: Sort scores descending → return Top-N (excluding query). Augmented with genre-filter secondary layer
sim(A,B) = (A · B) / (‖A‖ × ‖B‖)

TF-IDF vector dot product normalized by vector magnitudes
